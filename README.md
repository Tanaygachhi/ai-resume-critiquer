# AI Resume Critiquer

An AI-powered web application that analyzes resumes and provides improvement suggestions.

## Features
- Upload PDF resumes
- AI-based resume feedback
- Job-role specific suggestions

## Tech Stack
Python
Streamlit
Gemini API
PyPDF2

## How to Run

pip install -r requirements.txt
streamlit run main.py

Live demo: https://ai-resume-critiquer-v8xajooswgxmya9ynawtyf.streamlit.app/
